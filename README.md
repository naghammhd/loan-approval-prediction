# Loan Approval Prediction

This project demonstrates a machine learning workflow for predicting loan approval status using Python and scikit-learn.

## Features
- Data preprocessing
- Missing value handling
- Feature encoding
- Model training and evaluation
- Model comparison
- Prediction generation
- Data visualization

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Output
The project generates:
- Model performance comparison
- Prediction results
- Visualization charts
